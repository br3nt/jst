View Transitions

A View Transition is a browser API that animates the change between the old and new state of the DOM for you. You don't write the animation frame-by-frame — you just:

  1. call document.startViewTransition(callback),
  2. change the DOM inside the callback (here: swap the card's text + colour),
  3. the browser snapshots before & after and animates between them.

  — tick it and click Next to see the slide; untick it to see the same change happen instantly (no animation).
