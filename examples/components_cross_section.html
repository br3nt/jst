Modal / Dialog

command + <dialog>
Confirm action

Native <dialog> with focus trap, Esc, and backdrop — opened and closed entirely by command/commandfor. Zero JavaScript.

Accordion

CSS + HTML only
What is it?
Three <details name="acc"> — the shared name makes them mutually exclusive (Baseline). No script, no component.
How does it animate?
The marker rotates via CSS. details open/close is the browser's job.
What did JST add?
Nothing. This is the "just use HTML + CSS" outcome — and that's a win.

Dropdown menu

Popover + command

[popover] + command="toggle-popover". Outside-click and Esc dismissal are built in; the menu anchors to its button automatically.

Toast

JST component + command

A <jst-toaster> stack, fed by a custom --toast command; the message rides on the button's data-* (read from event.source). Auto-dismisses after 4s.

Combobox / Autocomplete

JST component

Type to filter; ↑↓ to move, Enter to pick, Esc to close. ARIA combobox/listbox roles.
